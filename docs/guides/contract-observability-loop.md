# Contract observability loop (tutorial)

**Contract observability** means watching JSON and MCP contracts continuously — not only at deploy time. DriftGuard combines **schema drift CI** in pull requests with hosted **API contract monitoring** when tools or APIs change in production.

**Status:** Hosted product tutorial — not maintained in the public OSS repository.

For the public onboarding path, start with [getting-started.md](../getting-started.md) and [policies/gate-ladder.md](../policies/gate-ladder.md). Run **mcp.json preflight** offline, then enable watches for **MCP tool catalog drift**.
